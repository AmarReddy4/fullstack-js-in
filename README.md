# fullstack-js-in

Building a full-stack application using MongoDB, Node.js, and React.js.

## Overview

This project is a hands-on exercise in full-stack JavaScript development, covering a REST API backend with Node.js/Express and a dynamic frontend with React.

### Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Getting Started

### Prerequisites

- Node.js (v12+)
- npm or yarn
- MongoDB (local instance or MongoDB Atlas)

### Installation

```bash
# Install server dependencies
npm install

# Install client dependencies
cd client
npm install
```

### Running the Application

```bash
# Start the backend server
npm run server

# In a separate terminal, start the React client
cd client
npm start
```
